*CHÚ Ý: Sử dụng IDE Dev-C++ để biên dịch chương trình (hoặc chạy file TuDienAnhViet.exe)

* TÁC GIẢ: Nguyễn Chí Cường (IUH)

* HƯỚNG DÂN SỬ DỤNG
	- Dùng các phím mũi tên để di chuyển lên xuống giửa các từ
	- Dùng phím F1 để tìm kiếm (sau khi bấm F1 thì nhấn tiếp Enter)
	- Dùng phím F2 để thêm từ mới (sau khi bấm F1 thì nhấn tiếp Enter)
	- Dùng phím F3 để sửa từ (sau khi bấm F1 thì nhấn tiếp Enter)
	- Dùng phím F4 để xóa từ (sau khi bấm F1 thì nhấn tiếp Enter)
	- Dùng phím ESC để thoát chức năng hiện tại

* THUẬT TOÁN - THƯ VIỆN SỬ DỤNG
	- Chương trình sử dụng cấu trúc dữ liệu con trỏ mảng struct
	- Sử dụng thuật toán Linear Seacrh để tìm kiếm từ
	- Sử dụng thuật toán Binary Search để tìm kiếm từ cần xóa, sửa, thêm (nếu không trùng)
	- Sử dụng thuật toán Quick Sort để sắp xếp dữ liệu
	- Sử dụng thư viện Windows.h để làm phần giao diện 

* CÁC FILE TRONG CHƯƠNG TRÌNH
	- File main.cpp là chương trình chính
	- File model.cpp || model.hpp lưu cấu trúc dữ liệu và các thuật toán liên quan tới dữ liệu
	- File view.cpp || view.hpp dùng để vẽ
	- File cosole.cpp || console.hpp dùng để giao tiếp với Cosole Window
	- File data.txt chứa dữ liệu
